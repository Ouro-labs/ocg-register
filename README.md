# Open Calling Graph Registry

Public static surface for MeshKit's Open Calling Graph registration and registry artifacts.

This repository intentionally contains only the public OCG registration website and public registry files. The private MeshKit implementation repository remains separate.

- Site root: `index.html`
- Whitepaper: `whitepaper.html` / `whitepaper.md`
- Registry index: `public/registry-index.json`
- Registry status: `public/registry-status.json`
- JWKS: `.well-known/ocg/jwks.json`
